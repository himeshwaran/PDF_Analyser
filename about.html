About PDF Analyzer

PDF Analyzer (MiniVirusTotal) is a lightweight web-based document inspection tool designed to detect and analyze suspicious components in files like PDF and DOCX. It focuses on identifying hidden scripts, embedded payloads, and metadata anomalies that indicate potential threats.

How It Works
  • Upload a document (PDF, DOCX, etc.)
  • The system uses open-source analysis tools such as pdfid and exiftool
  • It scans for JavaScript objects, hidden actions, and suspicious file structures
  • The scan results and a risk score are displayed instantly
Purpose

Developed as part of a cybersecurity research initiative to explore document-based malware detection and forensic analysis. The project demonstrates practical techniques used in threat hunting, file forensics, and defensive analysis against malicious documents.

Technologies Used
  • Frontend: HTML, CSS, JavaScript / Bootstrap
  • Backend: Python (Flask/Django)
  • Tools Integrated: pdfid, exiftool, oletools
